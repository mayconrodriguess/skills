# Skills

Esta pasta reúne skills organizadas por tema e fluxo de trabalho, com foco em uso prático de IA em cenários reais.

## Objetivo

Funcionar como uma biblioteca de skills reutilizáveis para acelerar tarefas técnicas, operacionais e de automação.

## Estrutura esperada de cada skill

Cada pasta pode conter:

- `SKILL.md`
  Arquivo principal da skill com identidade, objetivo, gatilhos e instruções de uso.

- `assets/`
  Templates, prompts auxiliares e materiais de apoio.

- `references/`
  Documentação, base conceitual, legislação, guias e referências técnicas.

- `scripts/`
  Scripts utilitários, automações e verificações relacionadas à skill.

- `agents/`
  Arquivos auxiliares para agentes especializados, quando necessário.

## Observação

As skills desta pasta foram organizadas para manter separação clara entre:

- conhecimento conceitual sobre prompting
- estruturas operacionais reutilizáveis
- fluxos especializados por domínio

O conteúdo conceitual e de guia permanece no repositório principal de prompts, enquanto esta pasta concentra a camada prática e executável.
